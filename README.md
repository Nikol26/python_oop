## Цель работы
Освоить объявление пользовательских классов.
Разобраться с инкапсуляцией (атрибуты экземпляра, закрытые поля).
Реализовать свойства (@property).
Переопределить магические методы (__str__, __repr__, __eq__).
Осознать разницу между атрибутами класса и экземпляра.

**Задумка**  
Создать "умного" студента, который:

~ Сам следит за корректностью данных (валидация)

~ Запрещает недопустимые действия (например, нельзя повысить отчисленного студента или повысить курс выше максимального)

~ Предоставляет удобный интерфейс через свойства (`@property`)

---

## Реализованный класс

**Student**

```python
class Student:
    MAX_COURSE = 6
    ACTIVE_STATUS = "active"
    EXPELLED_STATUS = "expelled"
Главный класс, моделирующий студента. Содержит всю основную логику работы с объектом студента.

---

## Атрибуты класса

- `MAX_COURSE` — максимальный курс (ограничение)
- `ACTIVE_STATUS` — активный студент
- `EXPELLED_STATUS` — отчисленный студент  

*Используются для логики и ограничений в программе.*

---

## Закрытые поля

- `_name` — имя студента  
- `_student_id` — ID студента  
- `_course` — курс  
- `_gpa` — средний балл  
- `_status` — статус  

*Это инкапсуляция — данные защищены и нельзя менять напрямую.*

---

## Свойства `@property`

- `name` — получение имени  
- `student_id` — получение ID  
- `course` — получение курса  
- `gpa` — чтение и изменение (через setter)  
- `status` — получение статуса  

*Позволяют безопасно работать с данными.*

---

## Валидация

- `_validate_name()` — проверка имени  
- `_validate_student_id()` — проверка ID  
- `_validate_course()` — проверка курса  
- `_validate_gpa()` — проверка GPA  
- `_validate_status()` — проверка статуса  

*Проверяют корректность данных (тип, диапазон, пустые значения).*

---

## Магические методы

- `__init__` — создание объекта  
- `__str__` — красивый вывод через print  
- `__repr__` — технический вывод  
- `__eq__` — сравнение объектов  

*Делают объект удобным для работы и вывода.*

---

## Бизнес-методы

- `promote()` — повышение курса  
- `expel()` — отчисление  

*Реализуют основную логику работы со студентом.*
